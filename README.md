# CursMLProiectFinal

Link catre dataset: https://www.kaggle.com/datasets/mexwell/fake-reviews-dataset

Dimensiune:
Datasetul complet conține aproximativ 40.000 de review-uri. Subsetul utilizat conține 4370.

Structura datasetului:
Datasetul are 4 coloane principale:

category – categoria produsului pentru care a fost scris review-ul (ex: Books, Kindle Store etc.)

rating – ratingul oferit produsului (de la 1 la 5 stele)

label – eticheta review-ului, care indică dacă review-ul este real (OR) sau fake (CG)

text_ – textul efectiv al review-ului scris de utilizator

Scopul datasetului:
Datasetul este utilizat pentru antrenarea unui model de machine learning care să poată detecta dacă un review este fals sau real pe baza textului acestuia.
